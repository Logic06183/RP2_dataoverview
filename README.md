# RP2 Clinical Dataset - Comprehensive Analysis

This repository contains a comprehensive statistical analysis and visualization of the RP2 clinical dataset.

## Overview

The analysis includes descriptive statistics, distributions, categorical data visualizations, and missing data analysis for multiple study profiles from Johannesburg-based research institutions.

## Viewing the Report

Visit the live site: [View Report](https://logic06183.github.io/RP2_dataoverview/)

Or open `index.html` locally in your browser.

## Study Profiles Included

- JHB_ACTG (015, 016, 017, 018, 019, 021)
- JHB_Aurum_009
- JHB_DPHRU (013, 053)
- JHB_EZIN (002, 025)
- JHB_JHSPH_005
- JHB_SCHARP (004, 006)
- JHB_VIDA (007, 008)
- JHB_WRHI (001, 003)

## Data Visualizations

Each study profile contains:
- Distribution plots
- Categorical data analysis
- Missing data heatmaps
