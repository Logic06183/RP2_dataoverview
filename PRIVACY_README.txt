DATA AGGREGATION STANDARDS:
================================================================================

All data in this package follows these privacy standards:

1. STATISTICAL AGGREGATION: All visualizations show aggregate statistics
   (means, medians, distributions) across groups of participants

2. MINIMUM CELL SIZES: No breakdowns shown for groups <5 participants

3. DE-IDENTIFICATION: All dates shifted, identifiers removed

4. SUMMARY STATISTICS ONLY: Individual observations never displayed

5. BINNED DATA: Continuous variables shown in ranges/bins to prevent
   re-identification
